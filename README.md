Назва: Smart Chess Calc — Розряди ФШУ

Опис:
Це спеціалізований веб-інструмент, розроблений для шахових арбітрів та тренерів Федерації шахів України (ФШУ). Додаток автоматизує процес розрахунку виконання спортивних розрядів та формування звітності за результатами шахових турнірів.

Основні можливості:

Обробка турнірних даних: Завантаження результатів турнірів через Excel-файли (.xlsx).

Автоматичний розрахунок: Обчислення середнього рейтингу, турнірних коефіцієнтів та визначення норм для здобуття розрядів (від юнацьких до КМС).

Картки гравців: Створення детальних персональних карток учасників із результатами кожної партії та статистикою.

Генерація сертифікатів: Автоматичне створення офіційних сертифікатів про виконання розрядних норм.

Експорт у PDF: Можливість збереження карток гравців та сертифікатів у форматі PDF для друку.

Адаптивний дизайн: Сучасний інтерфейс із підтримкою темної та світлої тем.
